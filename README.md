# Memory Reel - Digital Camera Carousel

A creative and fully responsive image slider interface styled as a vintage digital camera display screen, utilizing smooth asset swaps and explicit media control states.

## Features
- **Dual Control Workflow:** Supports both an automated auto-play playback cycle and localized manual track skips (Next/Previous).
- **Dynamic Pagination:** Interactive dot-matrix indices and real-time numeric indicators track the current item position accurately.
- **State Interoperability:** Play/Pause button seamlessly shifts underlying execution threads for precise carousel controls.

## Tech Stack
- HTML5
- CSS3 (Viewport clip-pathing, Relative Layouts, Transition Filters)
-  JavaScript (Array Iteration, Index Tracking Logic, Event Listeners)
